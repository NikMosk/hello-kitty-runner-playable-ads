# 🐱 Hello Kitty Runner — Playable Ad

**Ретровейв-раннер с пиксельной Hello Kitty, динамическим небом и синтвейв-музыкой**

## 🎮 Механика
- Тапай, чтобы Hello Kitty прыгала
- Собирай розовые бантики 🎀
- Уворачивайся от чёрных крестов
- 60 секунд на сбор максимального количества бантиков

## 🌞🌙 Цикл дня и ночи
- Солнце и луна движутся по кругу за горизонтом
- Плавная смена неба от рассвета до заката
- Ночью загораются окна в домах (случайные 30%)

## 📱 Поддерживаемые рекламные сети
- IronSource
- UnityAds
- Mintegral
- Moloco
- Liftoff

- **Вес:** < 2 МБ (с музыкой)

- **Адаптив:** ✅ (под любые экраны)
- **Трекинг событий:** ✅
